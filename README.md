# 醋风味数字化监测系统

> 基于多组学数据融合的中华传统调味品风味数字化检测与智能加工技术

## 项目简介

本项目针对中华传统调味品风味品质高度依赖经验判断、在线监测困难、品质稳定性不足等制约产业发展的核心问题，以镇江香醋这一具有千年历史的传统发酵食品为研究对象，综合运用代谢组学、感官组学、数据挖掘等多学科交叉方法，构建了一套基于多组学数据融合的"风味数字化检测-过程建模-智能决策"技术体系。

## 核心功能

### 1. 风味数字化检测
- 14种关键风味物质指标检测
- GC-MS/HPLC风味指纹图谱
- NIR快速筛查技术

### 2. 过程建模
- **五工序风味形成动力学模型**: 淀粉→葡萄糖→乙醇→乙酸→成品醋全流程追踪
- **AAF动力学模型**: 基于Luedeking-Piret模型，预测发酵周期缩短10-15%
- **陈酿演化模型**: Logistic模型描述乙酸乙酯、TMP等特征风味物质演化

### 3. 智能决策
- 翻醅时机量化推荐
- 陈酿条件优化建议
- 风味物质实时预测

## 核心模型

### AAF醋酸发酵动力学模型

采用分段Logistic拟合，创新性地实现发酵三阶段划分：

```python
from vinegar_model.aaf_kinetics import AAFModel

model = AAFModel()
state = model.get_state_at(8)  # 预测第8天状态
print(f"总酸: {state.total_acid} g/100mL")

# 翻醅建议
rec = model.recommend_turning(day=5, oxygen_lower=5.0)
print(f"需要翻醅: {rec['should_turn_today']}")
```

**验证结果**: R²(总酸)=0.91, R²(乙酸)=0.97

### 陈酿风味演化模型

```python
from vinegar_model.aging_kinetics import predict_at_age, age_to_state

# 预测5年陈酿风味状态
state = predict_at_age(60)  # 60个月
print(f"乙酸乙酯: {state.ethyl_acetate} μg/mL")
print(f"TMP: {state.tmp} μg/mL")
```
